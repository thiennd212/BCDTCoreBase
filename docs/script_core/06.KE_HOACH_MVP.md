## 4. Development Environment Setup

### 4.1. Prerequisites

| Tool | Version | Notes |
|------|---------|-------|
| Visual Studio 2022 | 17.8+ | Community/Professional |
| .NET SDK | 8.0+ | dotnet.microsoft.com |
| Node.js | 20 LTS | nodejs.org |
| SQL Server | 2022 | LocalDB / Express / Developer |
| Git | Latest | git-scm.com |
| Redis (optional) | Latest | Có thể dùng MemoryCache |

### 4.2. Quick Start Commands

```bash
# Clone & setup
git clone <repo-url>
cd BCDTCoreBase

# Database (chạy trong SSMS hoặc sqlcmd)
# Tạo database BCDT
# Chạy scripts: sql/v2/01 → 14

# Backend
cd src/BCDT.Api
dotnet restore
dotnet run
# → https://localhost:5001

# Frontend (terminal mới)
cd src/bcdt-web
npm install
npm run dev
# → http://localhost:3000

# Truy cập
# Login: admin / Admin@123
```
